# ARMA3ACE_ArtilleryComputer
ARMA3 ACE 자주포 포병 플레이어를 위한 고각/저각 발사 제원 계산기입니다.  
기본 사격통제컴퓨터는 편하지만 다소 부정확하며, 무조건 고각으로 계산하므로 즉각적인 화력지원이 어렵고, 저각 발사를 위해 사거리표를 보며 직접 계산하는것이 번거로워 만들었습니다.  

### 입력
- 거리
- 내 고도
- 타겟 고도
### 결과
- 필요 장약
- 발사각 밀값
- 착탄 소요 시간
